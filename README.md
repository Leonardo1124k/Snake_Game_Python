Jogo da Cobrinha criado totalmente em Python.  
Durante a criação deste game, consegui aprimorar minha prática no uso de funções (está em todo o código),  lógica de programação na elaboração de jogos, além de entender o uso de uma biblioteca tão relevante como PyGame.
